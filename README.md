🕹️ Apresentação do Jogo: Mario Jump

Este é um minijogo interativo de corrida infinita baseado na clássica mecânica de desviar de obstáculos do Super Mario. O objetivo é simples, mas desafiador: ajudar o Mario a correr pelo cenário e saltar no momento exato para não colidir com os canos verdes que surgem na tela.

O jogo conta com animações fluidas de nuvens ao fundo, efeito dinâmico de gravidade no pulo e um sistema de colisão preciso que congela a tela no milissegundo exato do impacto, transformando o Mario em sua clássica versão de Game Over. É um projeto leve, responsivo e excelente para testar os seus reflexos!
